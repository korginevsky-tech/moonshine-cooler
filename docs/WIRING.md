# Схема подключения компонентов

## Подключение ESP8266 NodeMCU

```
   ┌─────────────────────────────────────┐
   │       ESP8266 NodeMCU Board         │
   │                                     │
   │  Лева сторона (GPIO)                │
   │  ┌─────────────────┐                │
   │  │ D0              │ ← RST           │
   │  │ D1 (SCL I2C)    │ ← I2C Clock    │
   │  │ D2 (SDA I2C)    │ ← I2C Data     │
   │  │ D3              │                │
   │  │ D4 (GPIO2)      │ ← DS18B20      │
   │  │ D5 (GPIO14)     │ ← Реле компр.  │
   │  │ D6 (GPIO12)     │ ← Реле вентил. │
   │  │ D7 (GPIO13)     │                │
   │  │ D8 (GPIO15)     │                │
   │  │ 3.3V            │ ← питание      │
   │  │ GND              │ ← земля        │
   │  └─────────────────┘                │
   └─────────────────────────────────────┘
```

## Датчик температуры DS18B20

### Однопроводный режим (1-Wire)

```
     ┌─────────┐
     │DS18B20  │
     │  ┌─┐   │
     └──┤ ├───┘
        │ │
        1 2 3
        │ │ │
        │ │ └─── GND (черный провод)
        │ └───── D4/GPIO2 (желтый провод)
        │        ↓
        │     4.7k Ω подтяжка
        │        ↓
        └─────── 3.3V (красный провод)
```

**Таблица подключения:**
| Вывод DS18B20 | Назначение | Кабель |
|---|---|---|
| 1 (GND) | Земля | Черный |
| 2 (DQ) | Data | Желтый |
| 3 (VDD) | +3.3V | Красный |

**Резистор подтяжки:**
- Между DQ (вывод 2) и VDD (вывод 3)
- Значение: 4.7 кΩ
- Назначение: Стабилизация сигнала

### Множественные датчики (на одной шине)

```
ESP8266 D4 (GPIO2)
    │
    ├─── 4.7kΩ ───┬─── 3.3V
    │             │
    │          ┌──┴──┐
    ├──────────┤ DS1 │ (Жидкость)
    │          └─────┘
    │
    │          ┌─────┐
    ├──────────┤ DS2 │ (Пары)
    │          └─────┘
    │
    │          ┌─────┐
    ├──────────┤ DS3 │ (Вход)
    │          └─────┘
    │
   GND
```

## LCD/OLED дисплей I2C 16x2

### Подключение I2C

```
LCD I2C адаптер:

┌────────────────┐
│ GND  │ VCC │ SDA │ SCL│
└─┬────┴──┬──┴──┬──┴──┬─┘
  │       │     │     │
  │       │     │     └─ D1 (GPIO5) - SCL
  │       │     └─────── D2 (GPIO4) - SDA
  │       └──────────── 5V питания
  └──────────────────── GND

Перемычки на модуле I2C:
- VCC: +5V (или +3.3V если модуль 3.3V совместима)
- GND: GND
- SDA: D2 (GPIO4)
- SCL: D1 (GPIO5)
```

**Адреса I2C:**
- Большинство 16x2 модулей: **0x27** или **0x3F**
- Для определения: используйте I2C сканер

## Реле управления

### Реле компрессора (D5 / GPIO14)

```
┌─────────────────────────────────────┐
│     5V Реле модуль (2-канальное)   │
│                                     │
│  IN1 (Компрессор) ← D5 (GPIO14)   │
│  IN2 (Вентилятор) ← D6 (GPIO12)   │
│  GND ← GND                          │
│  VCC ← +5V                          │
│                                     │
│  COM1 ── Компрессор                │
│  NO1 ──                            │
│                                     │
│  COM2 ── Вентилятор                │
│  NO2 ──                            │
└─────────────────────────────────────┘
```

### Подключение к компрессору

```
220V Электросеть
├─ Фаза (коричневый) ─── COM1 (реле)
│                        
├─ Ноль (синий) ─────── Компрессор ─── NO1 (реле)
│
└─ Земля (желто-зел) ── Корпус компрессора
```

## Полная схема подключения

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  ESP8266 NodeMCU                                            │
│  ┌────────────────────────────────┐                         │
│  │ 3.3V ──┬─ LCD VCC (через 5V)   │                         │
│  │ GND ──┼─ LCD GND               │                         │
│  │       ├─ DS18B20 VDD (3.3V)    │                         │
│  │       └─ Реле VCC (5V)         │                         │
│  │                                │                         │
│  │ D1 (GPIO5) ─── LCD SCL (I2C)  │                         │
│  │ D2 (GPIO4) ─── LCD SDA (I2C)  │                         │
│  │ D4 (GPIO2) ─── DS18B20 DQ     │                         │
│  │ D5 (GPIO14) ── Реле IN1       │                         │
│  │ D6 (GPIO12) ── Реле IN2       │                         │
│  │ GND ──┬─ LCD GND              │                         │
│  │       ├─ DS18B20 GND          │                         │
│  │       ├─ Реле GND             │                         │
│  │       └─ 220V Ноль            │                         │
│  └────────────────────────────────┘                         │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## Список компонентов

| Компонент | Кол-во | Примечание |
|---|---|---|
| ESP8266 NodeMCU | 1 | Основной контроллер |
| DS18B20 | 2-3 | Датчики температуры |
| Реле модуль 5V | 1 | 2-канальное для компр. и вент. |
| LCD I2C 16x2 | 1 | Локальный дисплей |
| Резистор 4.7кΩ | 1 | Подтяжка для DS18B20 |
| USB кабель | 1 | Для программирования |
| Кабели DuPont | - | Для соединений |
| Конденсатор 100µF | 1 | Фильтр питания (опционально) |

## Рекомендации по электропитанию

- **ESP8266 + датчики**: 5V 1A
- **Реле модуль**: 5V 1A (отдельный БП рекомендуется)
- **Компрессор**: 220V отдельная сеть (через реле)

## Тестирование подключений

1. **ESP8266** - светится красный LED
2. **LCD дисплей** - синяя подсветка
3. **Реле** - слышна щелчок при срабатывании
4. **Датчики** - показания температуры в серийном мониторе

## Безопасность

⚠️ **ВАЖНО:**
- Используйте изолированные разъемы для 220V
- Никогда не касайтесь 220V проводов при включенной сети
- Используйте автомат 16A для защиты цепи компрессора
- Установите дифавтомат (УЗО) для защиты от утечек
